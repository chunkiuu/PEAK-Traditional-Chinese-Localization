# PEAK-Traditional-Chinese-Localization
《合力攻頂 PEAK》繁體中文化說明

本繁體中文語言檔是基於嗶哩嗶哩用戶「華麗的小檸檬啊」提供的簡體中文漢化版本進行轉譯而來。
我僅對原始簡體中文內容進行了繁體中文轉換。

原始簡體中文來源：
作者：華麗的小檸檬啊（Bilibili）

來源鏈接：https://space.bilibili.com/107001167

安裝方法：
將下載的語言檔（整個資料夾的內容）放入遊戲根目錄（與主程式 PEAK.exe 在同一層的資料夾）。

注意事項：
此為非官方翻譯，使用時請自行斟酌。
